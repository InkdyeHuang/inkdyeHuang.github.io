利用分布函数求概率

$P(a < X \leq b) = F(b) - F(a)$
$P(a \leq X \leq b) = F(b) - F(a-0)$
$P(a \leq X < b) = F(b-0) - F(a-0)$
$P(a < X < b) = F(b-0) - F(a)$
$P(x = a) = F(a) - F(a-0)$

根据概率密度函数求分布函数

高斯分布

概率密度函数:$f(x,\mu,\sigma) = \frac{1}{\sqrt{2\pi}\sigma}exp(-\frac{(x-\mu)^{2}}{2\sigma^{2}})$
概率分布函数:$F(X,\mu,\sigma) = \frac{1}{\sqrt{2\pi}\sigma}\int exp(-\frac{(t-\mu)^{2}}{2\sigma^{2}}) dt$
